INSTALL

Easy steps to install a Wordpress Theme

1. Unzip/Extract the theme
2. Upload the Theme folder to /wp-content/themes/
3. Go to Wordpress Admin > Presentation > Select the Theme

PLUGINS

Some of the plugins that goes best with Lorem-Ipsum Wordpress Themes
are included in the "plugins" folder!

You've to just upload the whole content of the "plugins" folder into your
/WordpressFolder/wp-content/plugins/.

Don't forget to activate the plugins.

Note: The theme is still independent of any plugin and it will behave normally
even without any plugins. The plugins just enhances theme and the enhanced features
will remain dormant without the plugins.
